# Pemrograman Berorientasi Objek (PBO)

Repository ini berisi kumpulan tugas, kuis, dan ujian dari mata kuliah **Pemrograman Berorientasi Objek (PBO)**. Proyek-proyek di dalamnya dikembangkan menggunakan bahasa pemrograman **Java** dengan fokus pada penerapan konsep-konsep OOP, pengelolaan basis data (H2/JPA), dan logika bisnis transaksi.

## Struktur Direktori

Proyek-proyek dikelompokkan ke dalam kategori berikut:

*   **[Exams](./Exams/)**: Berisi proyek Ujian Tengah Semester (Midterm) dan Ujian Akhir Semester (Final).
*   **[Fundamentals](./Fundamentals/)**: Berisi materi dasar seperti pengaturan lingkungan pengembangan (Environment Setup).
*   **[Quizzes](./Quizzes/)**: Berisi tugas-tugas kuis periodik selama perkuliahan.
*   **[Tasks](./Tasks/)**: Berisi kumpulan tugas mingguan (T01 - T06) yang mencakup implementasi sistem transaksi, manajemen akun, hingga penyimpanan persisten.

## Teknologi yang Digunakan
*   **Java SE**
*   **Maven** (Project Management)
*   **H2 Database** / **JPA (Java Persistence API)**
*   **JUnit** (Testing)
